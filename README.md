# Movie API
### Простое монолитное веб-приложение для каталога фильмов.
Приложение предоставляет собой API для взаимодествия с системой каталога фильмов. Приложение спроектировано согласно ТЗ, указанному ниже, но с некоторыми доработками.
#### <br> Перечень добавленных возможностей:
* поддержка фильтрации списка каталогов
* поддержка механизма генерации токена доступа с помощью рефреш-токена
* возможность смены пароля для пользователя

#### <br> Swagger документация доступна после запуска приложения по локальной ссылке:
    http://localhost:8080/swagger-ui/index.html#/
